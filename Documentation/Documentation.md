# Documentation



## User Endpoints


```
PUT Methods:

/Pawns/Move
    Flyttar en pawn på brädan
    

/Movefrombase
    Flyttar ut en pawn till brädan, respektive startposition.


POST Methods:

/Games
    Skapar upp ett game
    
```
